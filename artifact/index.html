OpenRouter serves "a model" from whichever provider is cheapest-and-up — at any quantization down to int4, dropping sampling params it doesn't support, silently. We audited 35 influential research repos flagged as routing model calls through it — in 32, its output reaches a published result. Here's who controls for it, who keeps it off the result path, and who's rolling the dice.

Read this correctly. “At risk” means the code leaves a known provider-routing corruption channel open and uncontrolled — not that any published number is wrong. We audited how each repo routes model calls (then adversarially re-checked every verdict against the source); we did not re-run experiments across providers to measure the actual delta. Highest real risk is on open-weight models on critical routes (output feeds a reported number, a training set, or a safety measurement).

Not all “safe” is the same. HANDLED = actually routes research calls through OpenRouter and controls for it (pinned provider, logged provenance) — a real positive example. OFF RESULT PATH = OpenRouter exists in the repo but no reported result depends on it. NO OR USAGE = no OpenRouter call site at all (a discovery false positive). Only HANDLED demonstrates anything about using OpenRouter well.

The denominator. The headline rate is AT RISK ÷ (AT RISK + HANDLED) — the repos whose OpenRouter output actually reaches a published result. OFF RESULT PATH and NO OR USAGE are excluded rather than counted as successes, because a repo that never routes a research call through OpenRouter has demonstrated nothing about using it well. Counting them instead would give the wider, weaker figure of 31/34.

“Possibly-impacted findings” are specific claims/figures we traced to an OpenRouter-routed call. They are hypotheses worth checking, not demonstrated errors — each carries its own “depends on OpenRouter” confidence. All 35 rows completed adversarial re-checking: a second agent re-read each paper and its code and was told to drop any cited figure it could not find in the real source. (No OpenRouter key was ever used to build this.)
